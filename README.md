# Vrinda Store Data Analysis

## Project Overview
This project analyzes Vrinda Store sales data to identify customer purchasing behavior, sales performance, revenue trends, and business insights. Interactive dashboards and visualizations were created to support data-driven decision-making and improve business performance.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Data Visualization

## Key Features
- Sales Performance Analysis
- Customer Demographics Analysis
- Order and Revenue Tracking
- State-wise Sales Analysis
- Channel-wise Sales Analysis
- Interactive Dashboard

## Key Insights
- Identified top-performing states and sales channels.
- Analyzed customer demographics and purchasing patterns.
- Evaluated monthly sales and revenue trends.
- Highlighted key factors contributing to business growth.
